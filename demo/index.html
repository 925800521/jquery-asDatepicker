Available options

You can pass these options as key/value object to $.datepicker() method.

API

show()


$('#calendar-api-show').datepicker('show');

APi

show


$('#calendar-api-show').datepicker('show');
Click to show calendar

hide()


$('#calendar-api-show').datepicker('hide');
Click to hide calendar

clear()

get_calendar()


$('#get-calendar').datepicker('get_calendar');
Returns the root element for the calendar.

get_input()


$('#get-inputr').datepicker('get_input');
Returns the input field.

get_value()


$('#get-value').datepicker('get_value');
Returns the currently selected date.

get_value(format)


$('#get-value-format').datepicker('get_value', 'yyyy-mm-dd');
String Returns the current date, formatted with the pattern given as the argument.

set_max()


$('#set-max').datepicker('set_max');

set_min()


$('#set-min').datepicker('set_min');
