jQuery.Datepicker

Examples

You can pass these options as key/value object to $.datepicker() method.

default single


<input type="text" id="calendar">
<script>
$(document).ready(function(){
  $("#calendar").datepicker();
});
<script>

range


<input type="text" id="calendar2">
<script>
$(document).ready(function(){
  $("#calendar2").datepicker({mode: 'range'});
});
<script>

multiple


<input type="text" id="calendar3">
<script>
$(document).ready(function(){
  $("#calendar3").datepicker({mode: 'multiple', calendars: '4'});
});
<script>

Available options

You can pass these options as key/value object to $.datepicker() method.

API

show()


$('#calendar-api-show').datepicker('show');
Click to show calendar

hide()


$('#calendar-api-show').datepicker('hide');
Click to hide calendar

multipleClear()


$('#calendar-api-multipleClear').datepicker('multipleClear');

getWrap()


$.datepicker('getWrap');

Returns the wrapper of the calendar.

getInput()


$.datepicker('getInput');
Returns the input field.

getDate()


$('#calendar-api-getDate').datepicker('getDate');

Returns the currently selected date.

getDate(format)


$('#calendar-api-getDate-format').datepicker('getDate', 'yyyy-mm-dd');

Returns the current date, formatted with the pattern given as the argument.

changeOptions()


$('#calendar-api-changeOptions').datepicker();
$('#calendar-api-changeOptions').datepicker('changeOptions', {mode: 'range'});

Sets one or more options for the datepicker.
